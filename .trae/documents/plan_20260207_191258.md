# Qt 仿微信聊天界面重构计划

根据您的详细需求，我将对现有的聊天界面进行深度重构。本次重构将移除旧的布局代码，严格按照您提供的技术栈和 UI/UX 规范进行实现。

## 1. 资源准备 (Resource Management)
- **目标**: 将所有图标（搜索、消息、联系人、设置、表情、文件等）和头像资源集成到 `resources.qrc` 中。
- **操作**:
  - 创建 `res/chat` 目录存放聊天相关资源。
  - 使用 Qt 内置标准图标作为临时占位（如果未提供自定义图标），或生成简单的 SVG/PNG 图标。
  - 确保代码中通过 `:/res/...` 路径加载资源。

## 2. 左侧侧边栏重构 (SessionSideBar)
- **类封装**: 创建 `ChatSideBar` 类 (继承 `QWidget`)，负责左侧所有 UI 和逻辑。
- **布局实现**:
  - **搜索栏**: `QWidget` + `QHBoxLayout`，自定义圆角搜索框 (`QLineEdit` + 搜索图标)。
  - **会话列表**:
    - 继承 `QListWidget`，禁用默认边框和滚动条样式。
    - 实现自定义 `QStyledItemDelegate` (`SessionItemDelegate`)，负责绘制头像（圆形）、昵称、消息预览和时间戳。
    - 使用 QSS 定义选中/悬停背景色。
  - **底部功能栏**: `QHBoxLayout`，包含消息、联系人、设置三个纯图标按钮 (`QPushButton`)。
- **数据结构**: 定义 `SessionItemData` 存储会话信息，通过 `QListWidgetItem::setData` 绑定。

## 3. 右侧聊天区域重构 (ChatArea)
- **类封装**: 创建 `ChatArea` 类 (继承 `QWidget`)，负责右侧所有 UI 和逻辑。
- **布局实现**:
  - **标题栏**: 固定高度 50px，居中显示当前聊天对象昵称。
  - **消息列表**:
    - 使用 `QScrollArea` (隐藏轨道) + `QVBoxLayout`。
    - 消息项使用自定义 `ChatBubble` 组件。
  - **输入栏**:
    - 固定高度 60px。
    - 包含表情、附件按钮。
    - 圆角输入框 (`QLineEdit`)。
    - 接收/发送按钮 (自定义样式和悬停效果)。
- **交互逻辑**:
  - 实现“接收”按钮逻辑：模拟接收对方消息。
  - 实现“发送”按钮逻辑：发送己方消息并清空输入框。
  - 消息列表自动吸底。

## 4. 消息气泡优化 (ChatBubble)
- **样式升级**:
  - **对方消息**: 绿色背景 (`#00B578`)，白字，圆角矩形（左侧直角，右侧圆角），右对齐。
  - **己方消息**: 灰色背景 (`#E5E6EB`)，黑字，圆角矩形（方向相反），左对齐。
  - **布局调整**: 增加发送者昵称显示（仅对方消息显示在气泡上方）。
  - **自适应**: 最大宽度 400px，高度自适应内容。

## 5. 主窗口整合 (MainWindow & ChatDialog)
- **布局调整**:
  - `ChatDialog` (或直接在 `MainWindow` 中) 使用 `QHBoxLayout`。
  - 设置固定比例：左侧 25% (Stretch 1)，右侧 75% (Stretch 3)。
  - 初始尺寸设定为 1046x932。
- **样式表 (QSS)**:
  - 编写全局 QSS，统一字体、滚动条样式、按钮 Hover 效果等。

  - `client/QmsrChat/sessionitemdelegate.h/cpp` (列表项绘制代理)
  - `client/QmsrChat/chatbubble.h/cpp` (气泡组件升级)
