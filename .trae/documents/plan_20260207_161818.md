# 核心界面代码生成计划 (纯 C++ 实现) - 优化版

根据您的需求及补充建议，我将移除现有的 `.ui` 文件，完全使用 C++ 代码构建仿微信布局，并实现核心的 `ChatBubble` 气泡控件。

## 1. 新增 ChatBubble 类 (消息气泡)
创建一个独立的 `QWidget` 子类，用于显示单条聊天消息。
- **功能**:
  - 支持 `Sender` (发送者) 和 `Receiver` (接收者) 两种模式。
  - 自绘圆角矩形背景 (绿色/白色)。
  - 包含头像 (`QLabel`) 和文本内容 (`QLabel`，支持自动换行)。
  - **[新增] 兜底头像绘制**: 若头像加载失败，绘制纯色圆形背景及名字首字母。
- **文件**: `client/QmsrChat/chatbubble.h`, `client/QmsrChat/chatbubble.cpp`

## 2. 重构 ChatDialog 类 (主聊天窗口)
完全移除 `chatdialog.ui`，在构造函数中通过代码实现布局。
- **主布局**: `QHBoxLayout` (Margins=0, Spacing=0)。
  - **左侧**: `QListWidget` (固定宽 250px，Stretch=0)。
  - **右侧**: 聊天区域容器 (Stretch=1)。
- **右侧布局 (Chat Area)**:
  - `QVBoxLayout`。
  - **顶部**: 标题栏 (`QLabel`)。
  - **中间**: `QScrollArea`。
    - **[新增] 自动吸底**: 连接 `verticalScrollBar` 的 `rangeChanged` 信号，确保新消息或图片加载时自动滚到底部。
  - **底部**: 输入区 (`QTextEdit` + `QPushButton`)。
    - **[新增] 键盘监听**: 安装 `eventFilter`，实现 **Enter 发送**，**Ctrl+Enter 换行**。
- **文件**: `client/QmsrChat/chatdialog.h`, `client/QmsrChat/chatdialog.cpp`
- **清理**: 删除 `client/QmsrChat/chatdialog.ui`。
